## Tukka Market 

A Prediction Market for Event based in Indian Sub-Continent. 

### The Initial Phase

We will be focusing on movie forecasting, Get the Latest Movies from the movie database API/ or other apis, and then use box office collection api for the `oracle` Event( Completion of the Prediction). 

We will have transctions, cancelling, and other trading features. When the Prediction is complete we will settle the books. 


### Research 
Learn more about how prediction market platforms like kalshi, and polymarket works.
